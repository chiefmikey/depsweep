# Context: DepSweep

_Date: 2026-03-01_
_Version: 1.0.0_

## What Is This

CLI tool for automated dependency cleanup with environmental impact reporting. Scans Node.js projects, identifies unused dependencies via Babel AST analysis, calculates carbon/energy/water savings.

## Architecture

```
src/
├── index.ts              (786 lines) - CLI entry, Commander.js, main logic
├── interfaces.ts         (104 lines) - TypeScript types
├── constants.ts          (515 lines) - Protected deps, env constants
├── helpers.ts           (1136 lines) - Dep analysis, env impact calculations
├── utils.ts              (650 lines) - AST analysis, file system, dep detection
├── performance-optimizations.ts (400 lines) - LRU cache, memory, monitoring
└── enhanced-environmental-calculations.ts (350 lines) - Detailed env calcs
```
